# 🚀 Milestone 2 – Advanced Security & Readability Dashboard

## 📌 Project Title
**Infosys Springboard – Policy Navigation Using AI (Milestone 2)**

---

## 📖 Description

In this milestone, the application was upgraded with stronger security features and an advanced readability analysis dashboard.

The focus of this milestone was to enhance security, improve the user interface, and introduce analytical capabilities for better document understanding.

Key improvements include OTP-based authentication, enhanced password security, login protection mechanisms, and a readability analysis dashboard that evaluates the complexity of uploaded documents.

---

## 🔐 Features Implemented

- OTP-based authentication for Signup and Password Reset  
- Secure 6-digit OTP generation with expiry  
- Limited OTP attempts for enhanced security  
- Password strength validation  
- Login rate limiting for security  
- Modern dark-themed UI for improved user experience  
- Readability Analysis Dashboard  

### Readability Metrics

The system evaluates uploaded documents using multiple readability algorithms:

- Flesch Reading Ease  
- Flesch-Kincaid Grade Level  
- SMOG Index  
- Gunning Fog Index  
- Coleman-Liau Index  

### Text Analysis

The application also calculates:

- Word count  
- Sentence count  
- Character count  
- Syllable count  

### Visualization

- Interactive readability gauges built using **Plotly**
- Easy-to-understand dashboard for document analysis

---

## 🛠 Technologies Used

- Python  
- Streamlit  
- SQLite  
- JWT (JSON Web Token)  
- Bcrypt  
- Plotly  
- PyPDF2  
- Textstat  
- SMTP (Email Integration for OTP Verification)
