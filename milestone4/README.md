# 📌 Project Overview

**PolicyNav** is an **AI-powered policy navigation platform** designed to simplify the process of understanding and interacting with complex policy documents.

Policy documents are often lengthy and difficult to navigate. This platform leverages **Artificial Intelligence and intelligent dashboards** to help users quickly explore policy information and gain insights through an intuitive interface.

### 🎯 Milestone 4 Objective

Milestone 4 enhances the platform with advanced features including:

- **Admin Management Dashboard**
- **User Dashboard & Profile Personalization**
- **System Analytics & Visualization**
- **User Feedback Analysis**
- **Data Export Capabilities**

These additions significantly improve **platform monitoring, user control, and system transparency**.

Milestone 4 builds upon and integrates all capabilities developed in **Milestone 3**.

# 🛡 Admin Dashboard

The **Admin Dashboard** functions as the **central control system** for the platform.  
It allows administrators to monitor system activity, manage users, and analyze platform usage.

---

## 🔑 User Management

Administrators can manage platform users through the following actions:

✔ Promote users to **Admin role**  
✔ **Lock or unlock** user accounts  
✔ **Delete users** from the system  

This ensures **secure role-based access control**.

---

## 📊 Activity Monitoring

The dashboard allows administrators to track system usage in real time.

Monitoring features include:

- 👥 **Active users**
- 📜 **Complete system activity history**
- 📈 **Platform interaction tracking**

---

## 📈 Data Visualization

Interactive charts provide insights into system behavior and user interaction.

Analytics include:

- 🤖 **Most frequently used AI models**
- 🌐 **Most used languages**
- ⭐ **Most popular platform features**

---

## 💬 Feedback Analysis

User feedback is analyzed using a **WordCloud visualization**.

This feature helps administrators quickly identify:

- Common concerns
- Frequently requested improvements
- Popular user suggestions

---

## 📥 Data Export

Administrators can export platform data for reporting and analysis.

Exportable data includes:

- User feedback
- Activity history logs
- User account information

---

# 👤 User Dashboard

The **User Dashboard** improves **account security, customization, and user experience**.

---

## 🔒 Security Features

Users can manage their account security by:

- Updating their **email address**
- Changing their **account password**

---

## 🧑‍🎨 Profile Personalization

Users can personalize their accounts through:

- Uploading a **profile avatar (Display Picture)**
- Updating their **profile details**
